# Software Testing Examples

All testing examples and documentation are on the feature branches:

## To Access Testing Examples:

### Vulnerable Code (Educational):
```bash
git checkout vulnerable-version
```

### Secure/Fixed Code:
```bash
git checkout secure-version
```

## What's on Each Branch:

- **vulnerable-version**: Intentionally vulnerable Python and Java code for learning
- **secure-version**: Both vulnerable and fixed versions with full documentation

Master branch is a documentation hub only. Switch branches to access actual code and detailed documentation.
